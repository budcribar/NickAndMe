## Picture-book structure (use it)
- Filenames like page_001_render.png / page_NNN_*.png encode page order (NNN = page number).
- In many illustrated picture books, spreads alternate: **odd pages = art**, **even pages = text**.
- Prefer **odd** page numbers for character art unless the name clearly marks text-only (text, text_page, text_heavy).
- Prefer **cover** only for the **title / hero** character when the key/description matches the main subject of the book.
- **Supporting cast** (secondary animals/people not the hero): often appear on **one** mid/late illustrated page, not the cover and not the first few pages. Prefer later odd pages for them.
- If the character is a **rabbit / bunny** (or other non-hero animal) and the hero is a dog (or other species), do **not** rank early dog-hero pages just because they are first in the list.

## Matching
- Use character_key + description species/role (dog vs rabbit, parent vs child, etc.).
- Prefer illustrated basenames over text-only basenames.

- If unsure for a supporting character, prefer a single plausible mid/late illustrated page over inventing presence on page 1–3.
